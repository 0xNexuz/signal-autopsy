# Signal Autopsy

Pre-trade failure intelligence for AI trading agents.

Signal Autopsy turns an agent's proposed trade into a replayable pre-mortem: market state, execution pressure, contradiction, guardrails, and an audit trail before live routing is allowed.

Live site: https://signal-autopsy.vercel.app

## Run Locally

Open `index.html` in a browser.

## Deploy

This is a static site and can be deployed directly to Vercel.
